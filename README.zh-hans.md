# 星河

一个Linux的**动态壁纸引擎** （开发中）

[English](https://github.com/SMLJerry/linux-dynamic-wallpaper-engine/blob/main/README.md)

### 特性

- [ ] 将视频作为动态壁纸
- [ ] 支持GNOME3桌面
- [ ] 支持KDE5.x桌面
- [ ] 支持XFCE4桌面
